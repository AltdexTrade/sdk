# 🚀 Altdex

**One AI bot. Two exchanges. Every trade streamed live.**  
Hunting high-probability breakouts across **ALTDEX DEX** and **Binance Futures** in real-time.

---

## 🧠 Overview

**Altdex** is an AI-powered trading system designed to detect breakout opportunities and execute trades seamlessly across multiple exchanges.

By combining real-time data analysis, smart execution logic, and live trade streaming, Altdex delivers a transparent and efficient trading experience for modern crypto markets.

---

## ⚡ Features

- 🤖 AI-driven breakout detection  
- 🔄 Multi-exchange trading (DEX + Futures)  
- 📡 Live trade streaming  
- ⚡ Low-latency execution  
- 📊 Real-time market monitoring  
- 🔐 Secure wallet & API integration

## 🏗️ Project Structure

```
altdex/
├── bot/                # Core AI trading logic
├── strategies/         # Breakout detection strategies
├── exchange/           # Exchange integrations (DEX + Binance)
├── streaming/          # Live trade streaming system
├── utils/              # Helper functions
├── config/             # Configuration files
├── logs/               # Trading logs
├── .env.example        # Environment variables template
├── requirements.txt    # Dependencies
└── main.py             # Entry point
```

---

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/altdex.git
cd altdex
pip install -r requirements.txt
```

---

## 🔑 Configuration

Rename `.env.example` to `.env` and fill in your credentials:

```
BINANCE_API_KEY=
BINANCE_API_SECRET=
WALLET_PRIVATE_KEY=
RPC_ENDPOINT=
```

---

## ▶️ Usage

```bash
python main.py
```

Bot will start scanning markets, detecting breakouts, and executing trades automatically.
